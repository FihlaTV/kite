Kite API

Shape

Represents a shape that is composed of one or more subpaths. new kite.Shape() will create an empty shape, which can then have drawing commands (like moveTo or lineTo) applied to it.

moveTo( x, y )

Moves the current position to the (x,y) location, without drawing a line. See context.moveTo
